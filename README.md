# 🇧🇷 Exercícios de Portugol — 2020

Repositório histórico com os primeiros exercícios de lógica de programação que desenvolvi em **outubro/novembro de 2020**, utilizando a linguagem **Portugol** no [Portugol Studio](https://univali-lite.github.io/Portugol-Studio/).

Esses exercícios marcam o início da minha jornada na programação — antes de aprender Python, Java ou qualquer outra linguagem.

---

## 📁 Estrutura

```
exercicios-portugol-2020/
├── semana-01/   → 10 exercícios — Sequência e operações básicas
├── semana-02/   → 11 exercícios — Estruturas condicionais (se/senão)
├── semana-03/   →  4 exercícios — Estruturas de repetição (enquanto/para)
└── semana-04/   →  1 exercício  — Reentrega / revisão
```

---

## 🧠 Conteúdo por semana

### Semana 01 — Sequência e operações básicas
Algoritmos com entrada, processamento e saída simples, sem desvios condicionais.
Exemplos de problemas resolvidos:
- Cálculo de salário com comissão de vendedor
- Percentual de votos brancos, nulos e válidos em eleições
- Conversões e cálculos matemáticos do cotidiano

### Semana 02 — Condicionais (`se` / `senão se` / `senão`)
Problemas que exigem tomada de decisão com múltiplos caminhos.
Exemplos de problemas resolvidos:
- Cálculo de IPVA por categoria de veículo (com distribuição entre governo, município e FUNDEB)
- Verificações com múltiplas condições encadeadas

### Semana 03 — Repetição (`enquanto` / `para`)
Algoritmos com laços de repetição e menus interativos.
Exemplos de problemas resolvidos:
- Exibição de números pares, ímpares ou sequência completa com menu de opções
- Problemas que exigem validação de entrada com laço

### Semana 04 — Revisão
Reentrega ou refinamento de exercício da semana 01.
